У цьому завданні вам потрібно буде створити базу даних,
створити таблицю в цій базі даних, вставити дані в таблицю,
вибрати дані з таблиці та видалити деякі записи.

Створення бази даних: Створіть базу даних з назвою my_database.

Створення таблиці:Створіть таблицю з назвою users в базі даних my_database,
яка буде містити наступні поля:
- id (цілочисельний тип, автоінкрементний, первинний ключ),
- name (рядковий тип),
- age (цілочисельний тип),
- email (рядковий тип).

Вставка даних:Вставте наступні дані в таблицю users:
- name: "John", age: 30, email: "john@example.com"
- name:"Alice", age: 25, email: "alice@example.com"
- name:"Bob", age: 35, email: "bob@example.com"

Вибірка даних:Виберіть всі записи з таблиці users та виведіть їх.

Видалення даних:Видаліть користувача з ім'ям"Bob" з таблиці users.

Перевірка правильності виконання:Переконайтеся, що дані були
вставлені правильно та виведені після вибірки.

Переконайтеся, що користувач з ім'ям "Bob" був успішно видалений.

Запити зберегти в файлі та зробити скріншоти результатів виконная запитів.
Залийте файл та скріншоти в свій GitHub репозиторій, посилання на який зазначте в LMS
