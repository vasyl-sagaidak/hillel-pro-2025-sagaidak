Атомарные операции, это такие операции которые не могут быть прерваны
планировщиком потоков. Выполнение такой операции должно быть выполнено
полностью (если она уже происходит), либо еще не быть выполнено.

Атомарными операциями в Java являются операции присваивания со
всеми примитивными типами кроме long и double,
а так-же атомарными операциями является присваивание ссылок.

1) = with primitive types (except long and double)
2) = with references
