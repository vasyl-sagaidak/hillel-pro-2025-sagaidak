У цьому завданні вам потрібно буде реалізувати принцип Expert для надання відповідальності
належним об'єктам.

==================
1.Створіть класи:

Створіть клас User, який представляє користувача системи.
Створіть клас Address, що містить інформацію про адресу користувача.

==============================
2.Налаштуйте відповідальність:

Надайте класу User відповідальність за зберігання адреси користувача, використовуючи
об'єкт класу Address.
Клас User буде мати методи для роботи з адресою, такі як setAddress() та getAddress().

=================================
3.Створіть клас для демонстрації:

Створіть клас Main, який містить метод main().
У методі main() створіть об'єкт класу User та об'єкт класу Address.
Встановіть адресу для об'єкта користувача за допомогою методу setAddress().
Виведіть адресу користувача на консоль, використовуючи метод getAddress().

=================================
4.Перевірте правильність роботи:

Переконайтеся, що адреса користувача належить об'єкту класу User.
Переконайтеся, що клас User відповідає за доступ до адреси та виконує необхідні дії з нею.
5.Залийте виконаний проєкт у свій GitHub репозиторій, посилання на який зазначте в LMS
