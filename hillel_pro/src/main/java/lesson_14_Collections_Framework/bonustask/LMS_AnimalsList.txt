Опис задачі: В проекті, який складається з декількох класів,
має бути реалізовано функціонал виведення нумерованого списку
тваринок та тваринки за певним індексом за допомогою можливостей
Interface List та Class ArrayList.
Маємо такий перелік тваринок:

cat
dog
mouse
fox

Виведення має бути таким:

(1) cat (2) dog (3) mouse (4) fox
fox is in index 3

Доопрацюйте функціонал класу DataHandler
