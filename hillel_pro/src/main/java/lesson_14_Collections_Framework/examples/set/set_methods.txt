Интерфейс Set и его самые популярные реализации:

Class HashSet
Class LinkedHashSet
Class TreeSet

Свойства:

- хранить null можно;
- дубликаты значений нельзя;

Методы:

add(E e) - добавляет элемент в сет, если внутри такого еще нет.

addAll(Collection<? extends E> c) - добавляет все элементы определенной коллекции в набор,
                                    если они еще не присутствуют в нем. Если добавляется набор в набор, то операция
                                    addAll фактически изменяет набор таким образом, что его значение является обьеди-
                                    нением двух наборов.

clear() - удаляет все элементы из набора (очищает набор)

сontains(Object o) - возвращает true, если набор содержит указанный элемент.

containsAll(Collection<?> c) - возвращает true, если набор содержит все элементы указанные в коллекции. Если набор
                           содержит указанный набор, этот метод возвращает true, если он явл.подмножеством этого набора.

isEmpty() - возвращает true если набор не содержит элементов (пустой).

remove(Object o) - удаляет указанный элемент из набора, если он там присутствует.

removeAll(Collection<?> c) - удаляет из набора все его элементы, которые содержатся в указанной колекции. Если удаля-
                             ется набор из набора, эта операция фактически изменяет модифицированный набор таким образом
                             , что его значение является ассиметрической разницей набора двух наборов.

retainAll(Collection<?> c) - оставляет только те элементы из набора, которые содержатся в указанной коллекции. Тоесть,
                             удаляет из этого набора все его элементы, которые не находятся в указанной коллекции.
                             Если элементы для оставления, имеющиеся в указанном наборе, то эта операция вактически
                             изменяет набор таким образом, что его значение является пересечением двоих наборов.

size() - возвращает количество элементов в наборе.
