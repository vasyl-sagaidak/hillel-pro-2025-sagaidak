Завдання:

Реалізувати фільтрацію та обробку даних за допомогою Stream API

Опис:

Завдання передбачає створення класів для обробки та фільтрації
списку продуктів з використанням Stream API.
У цьому контексті використовується клас Product,
що містить інформацію про продукти, а також інтерфейс ProductFilter та
клас PriceFilter, які дозволяють виконати фільтрацію продуктів за ціною.

Створіть клас Product який містить дані про продукт,
такі як назва (name) та ціна (price).
Клас також надає методи для отримання цих даних та перевизначений
метод toString для представлення об'єкта у вигляді рядка.
