У цьому завданні вам потрібно буде виконати ряд завдань з використання
JDBC для роботи з базою даних.

1.Створіть базу даних (Використайте наприклад PostgreSQL):

a. Створіть базу даних з назвою company.

b. Створіть таблицю employees, яка буде містити поля:

id (типу INTEGER, AUTO_INCREMENT, PRIMARY KEY),
name (типу VARCHAR),
age (типу INTEGER),
position (типу VARCHAR),
salary (типу FLOAT).

2.Підключіться до бази даних за допомогою JDBC:

a. Створіть клас DatabaseConnector, який буде містити методи для підключення до
бази даних, виконання запитів та закриття з'єднання.

b. Реалізуйте методи для підключення до бази даних та виконання запитів
(для створення таблиці та вставки даних).

3.Створіть методи для роботи з базою даних:

a. Створіть клас EmployeeDAO, який буде містити методи для додавання, оновлення,
видалення та отримання інформації про співробітників з бази даних.

b. Реалізуйте методи для додавання, оновлення, видалення та отримання
інформації про співробітників.

4.Перевірка правильності виконання:

a. Створіть клас Main, в якому викличте методи для додавання, оновлення,
видалення та отримання інформації про співробітників та виведіть результати на екран.

b. Переконайтеся, що всі методи працюють коректно та виводять очікуваний результат.

5.Залийте файл та скріншоти в свій GitHub репозиторій, посилання на який зазначте в LMS
