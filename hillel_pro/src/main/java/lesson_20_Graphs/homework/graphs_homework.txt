У цьому завданні вам потрібно буде створити клас для роботи
з графами та реалізувати базові операції над ними.

Створіть клас для виконання завдання:

a. Створіть клас Graph, який буде представляти граф та містити методи для додавання вершин,
додавання ребер, видалення вершин, видалення ребер, а також для перевірки чи існує вершина чи ребро у графі.

Реалізуйте методи для роботи з графом:

a. Метод addVertex(int vertex), що додає вершину до графу.
b. Метод addEdge(int source, int destination), що додає ребро між двома вершинами.
c. Метод removeVertex(int vertex), що видаляє вершину з графу.
d. Метод removeEdge(int source, int destination), що видаляє ребро між двома вершинами.
e. Метод hasVertex(int vertex), що перевіряє чи існує вершина у графі.
f. Метод hasEdge(int source, int destination), що перевіряє чи існує ребро
   між двома вершинами у графі.

Створіть методи, щоб продемонструвати роботу з графом:

a. Створіть граф та додайте до нього вершини та ребра.
b. Перевірте роботу методів додавання, видалення та перевірки вершин та ребер.
