1.1 Cоздать класс PetrolStation.

1.2 Класс PetrolStation содержит свойство с именем amount.
    Это свойство отражает количество топлива на станции.

1.3 В классе PetrolStation реалиовать метод doRefuel.
    В методе есть входящий параметр - количество топлива для одной заправки.

1.4 Одна заправка длится 3-10 секунд. После окончания времени ожидания, выполняется
    операция вычитания топлива (количество, которое приходит в данный метод, от общего
    запаса на PetrolStation.

1.5 Заправочная станция может одновременно обслуживать не более трех автомобилей,
    если очередь больше, то остальные ждут.

1.6 Использовать очередь для внутренней реализации.

Реализация через многопоточность, взять очередь которую дали на занятии,
либо другую какую-то.
