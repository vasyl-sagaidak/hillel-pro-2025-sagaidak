1) Создать базовый тип Fruit
2) Создать типы Apple, Orange, которые наследуют Fruit
3) Создать тип Box. В коробку можна складывать фрукты, но только фрукты одного типа.
   Например: Яблоки с апельсинами мешать в одной коробке нельзя.

4) Реализовать метод add() - добавления фрукта (1шт) в коробку.
5) Реализовать метод add() - добавление фрукта (множ.) в коробку используя массив.
6) Реализовать метод getWeight(). Метод учитывает вес коробки на основании веса одного фрукта и количества
   фруктов в коробке.
   Вес яблока - 1.0F, апельсина - 1.5F.
7) В классе Box реализовать метод boolean compare(Box box). Метод делает сравнение данной коробки с переданной как параметр.

    ВАЖНО! Можно сравнивать коробки содержащие разные фрукты.
    Так, коробки с яблоками можно сравнивать с коробками с апельсинами и.т.п.

8) Реализовать метод merge(). Метод осуществляет пересыпку с переданной в качестве парамера коробки в имеющуюся.
   ОПЯТЬ ЖЕ: Следует учитывать что запрещается хранить в одной коробке фрукты разного типа.
